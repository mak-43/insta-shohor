How does javascript works

Javascript used for a client-site web application which is single thread high level interpreted and scripting language. Modern browsers use a technology known as Just-In-Time (JIT) compilation, which compiles JavaScript to executable bytecode just as it is about to run. V8 javascript engine is used in Google chrome.

Differences between local and session storage
Local Storage
  • The data does not expire. It remains after the browser restart and even OS reboot.
  • Storage Capacity 5MB/10MB(Browser Dependent)
  • Shared between all tabs and windows from the same origin.
  • The client can only read local storage
    • Session Storage
  • The sessionStorage exists only within the current browser tab.
  • Storage Capacity 5MB
  • Another tab with the same page will have a different storage.
  • The client can't read session storage

    • What does JS event loop do

    Event loops are the secret behind JavaScript asynchronous programming. JS performs all operations in a single thread, but with some smart data structures like multithreading.
