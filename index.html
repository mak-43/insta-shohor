How does javascript works

Javascript is a client site web application which is single thread high level interpreted and scripting language. Everything in javascript happens in inside an execution context. Execution context is just like a big box or container. Execution Context seperated by memoey component and code component.In Memoey component variables store as key,value pair.In code component part full code is executed line by line at a time from top to bottom. Basically, JS is a synchronous single-threaded language. Js workes in Browser via v8 Engine
